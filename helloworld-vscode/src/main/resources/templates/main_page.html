Estructura de Datos

Esta asignatura pretende ser un curso introductorio a las estructuras de datos clásicas tratadas con el enfoque de tipos abstractos de datos (TAD), y al desarrollo metodológico de programas que usen dichas estructuras. Al terminar el curso, los alumnos deberán conocer las propiedades, el funcionamiento y las posibles implementaciones de las principales estructuras de datos, y saber cuándo y cómo utilizarlas para la resolución de problemas.

Desarrollo de Aplicaciones Web

El objetivo de la asignatura de “Desarrollo de Aplicaciones Web” es ofrecer al alumno una visión general del desarrollo de las aplicaciones diseñadas para ser utilizadas a través de navegadores Web, ya sea en ordenadores personales o en dispositivos móviles como teléfonos inteligentes y tabletas. La asignatura presenta las características más importantes del desarrollo de aplicaciones Web y proporciona al alumno una panorámica general de las tecnologías, arquitecturas y protocolos más utilizados en el desarrollo de este tipo de aplicaciones. Su principal objetivo es que los alumnos apliquen estos conocimientos en la construcción de una aplicación web real con unas funcionalidades similares a las que podría tener cualquier web profesional.

  • Web
